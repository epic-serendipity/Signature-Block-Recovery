# Quick Start

1. **Download & Install**
   - **Windows:** run the MSI and use the desktop shortcut.
   - **macOS:** open the DMG and drag the app to Applications.
2. **Launch**
   - On first run, select your PST file(s) and choose an index location.
   - Extraction starts automatically.
3. **Search & Export**
   - Query the index, adjust filters, then export to CSV/JSON/Excel.
